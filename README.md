# Project-Spotify
Spotify Clone 🎵 – A fully responsive, mobile-friendly music player built with JavaScript. Play songs from folders, search tracks, navigate with next/previous controls, and enjoy smooth animations with a hamburger toggle for mobile. Perfect for learning JS &amp; responsive UI.
